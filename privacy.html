本隐私政策由 Aloosh IQ 提供，适用于由 Aloosh IQ 开发的移动应用 Aesthetic Color Wallpapers 4K（以下简称“本应用”）。本隐私政策旨在详细说明 Aloosh IQ 如何处理您的个人信息。

1. 我们收集的信息

当您使用由 Aloosh IQ 开发的本应用时，我们可能会收集以下信息以提供核心服务：

  • 您的设备互联网协议地址（如 IP 地址）。
  • 设备识别码： 包括 华为广告标识符 (OAID) 和 Google 广告 ID (GAID)。这些信息对于提供个性化服务至关重要。

2. 数据存储与缓存管理

Aesthetic Color Wallpapers 4K 包含 “清除缓存” (Clear Cache) 功能。此功能允许用户手动删除存储在设备本地的临时图片文件。此类操作完全在本地执行，Aloosh IQ 不会收集或上传任何与您的本地文件相关的数据。

3. 第三方 SDK 与广告透明度

为了支持应用运营并由 Aloosh IQ 提供免费内容，我们集成了以下第三方服务：

  • Yandex (Yango) 广告联盟： 用于展示 个性化广告。用户可以随时通过应用内的 关闭按钮 或系统设置跳过或关闭广告。
  • 华为 HMS Core： 在华为设备上通过 OAID 安全地处理广告数据。

4. 您的权利 (PIPL)

根据中国《个人信息保护法》(PIPL)，您有权查阅、复制、更正或删除您的个人信息。您可以随时通过 tekinmkmk@gmail.com 与我们联系以行使这些权利。

5. 儿童隐私保护 (PIPL 准则)

Aloosh IQ 非常重视对未成年人个人信息的保护。本应用不直接面向 14 周岁以下的儿童。我们不会有意收集未满 14 周岁儿童的个人信息。如果您认为您的孩子在未经监护人同意的情况下向 Aloosh IQ 提供了信息，请立即通过 tekinmkmk@gmail.com 联系我们，我们将采取措施删除相关数据。

6. 联系方式

如果您对 Aloosh IQ 的隐私做法有任何疑问，请通过 tekinmkmk@gmail.com 与我们联系。
